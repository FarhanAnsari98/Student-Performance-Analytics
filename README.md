# Student Performance Analytics System

A modern web-based Student Performance Analytics System developed using **Next.js**, designed to simplify academic performance tracking and visualization for educational institutions.

The system provides separate portals for **Admin, Teachers, Students, and Parents**, enabling efficient management and monitoring of student academic records.

---

## Features

- Secure Login System for:
  - Admin
  - Teacher
  - Student
  - Parent

- Admin Dashboard
  - Manage users and system records
  - Monitor overall system activity

- Teacher Portal
  - Add and update student marks
  - Manage attendance records
  - Generate academic analytics

- Student Portal
  - View marks and attendance
  - Access performance reports
  - Analyze progress using charts and graphs

- Parent Portal
  - Monitor child academic performance
  - View attendance and progress reports

- Data Visualization
  - Bar Charts
  - Pie Charts
  - Performance Analytics Dashboard
